# Time Awareness Tool — Agent 时间感知工具

## 概述

给虞晚 agent 增加一个 `get_time_context` 工具，让她能感知当前时间、时段、日期、节日/节气，从而自然调整对话内容（不会一大早说晚饭，深夜聊早餐）。

## 改动范围

单文件：`daiyu_agent/agent/tools.py`

## 函数设计

`get_time_context()` — 无参数，返回格式化时间上下文字符串。

### 返回值格式

```
现在是周三下午。2026年5月23日 星期三 15:42。临近端午节（还有8天）。
```

- 首句：时段标签（斜体，作为语气提示）
- 第二行：完整日期时间
- 第三行：节日/节气提示（仅距离7天内有活动时显示，无则省略）

### 时段映射

| 小时范围 | 标签 |
|----------|------|
| 0-5 | 深夜 |
| 5-8 | 清晨 |
| 8-11 | 上午 |
| 11-13 | 中午 |
| 13-17 | 下午 |
| 17-19 | 傍晚 |
| 19-23 | 晚上 |
| 23-24 | 深夜 |

### 节日数据

硬编码中国传统节日（农历对应的公历日期取每年近似值，±1~2天）：

- 春节 (农历正月初一)
- 元宵节 (农历正月十五)
- 清明节 (公历4月4-6日，取4月5日)
- 端午节 (农历五月初五)
- 七夕 (农历七月初七)
- 中秋节 (农历八月十五)
- 重阳节 (农历九月初九)
- 冬至 (公历12月21-23日)

二十四节气（按公历近似日期，24个）。只显示距离当前日期7天以内的节日/节气。

### LangChain 工具注册

使用 `@tool` 装饰器，加入 `AGENT_TOOLS` 列表：

```python
from langchain_core.tools import tool

@tool
def get_time_context() -> str:
    """获取当前时间上下文：时段、日期、临近节日/节气"""
    ...
```

## 技术决策

- **零外部依赖**：不引入 `datetime` 以外的库，不调外部 API
- **节日硬编码**：中国节日不像西方节日那样公历固定，但用硬编码近似值够用（偏差1-2天，用户不会在意）
- **7天阈值**：避免每次对话都带无关的节日信息

## 不做什么

- 不做农历换算（引入第三方库太重，硬编码够用）
- 不做时区配置（取系统本地时间）
- 不做节假日 API 调用
- 不修改 agent/engine.py、routers/chat.py、config.py
